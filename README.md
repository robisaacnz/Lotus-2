# Lotus Turbo Challenge 2
Lotus Turbo Challenge 2 is a racing game developed by Magnetic Fields and published by Gremlin Interactive, released in 1991.

## Controls
←					Left
→					Right
↑					Up
↓					Down
Right ⎇ 	Fire

⎇+⌘+Q		Quit
